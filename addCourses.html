Instructions ▼

  • To add a class, just type the subject and course ID
    • For example, to add "CS 4550: Web Development", I would type "CS4550".
  • Next, hit the "Enter" key or press the arrow button
    • If we find a course matching your input, we'll add it!
    • If the course has any corequisites, you'll see an option to add them
  • Finally, select a semester and hit "Submit"!
