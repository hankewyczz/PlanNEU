Instructions

  • First, select a semester from the dropdown selector
  • To add a class, simply type its subject and course ID
    • For example, if I wanted to add the course "CS 4550: Web Development", I would type "CS4550".
  • Next, hit the "Enter" key or press the button
    • If we can find the course, you'll get a 'Success!' message, and the course will show up in the "Courses Selected" section below
    • If the course has any corequisites, you'll see an option to add them
  • Once you've added all the courses, simple press "Submit"!
